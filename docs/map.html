Routing sources and limits

Driving estimates use the public OSRM Route service for list-order legs and the OSRM Table service for drive-time sorting. Results are estimates from OpenStreetMap-based routing and are not traffic-aware. Google Maps route export uses Google Maps URLs; the mobile route link opens the first five mapped stops. Google route chunks create sequential Google Maps links for every mapped stop, while KML and CSV include every saved permit with details.
